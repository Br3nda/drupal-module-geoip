Database Files
--------------------
Before using this module, you will need to download one of the GeoLite database
files. These files are provided by MaxMind as gzipped files.  You will need to
gunzip these data files before using them.

To be installed in the data directory:

- GeoLite Country binary format database
  - Information: http://www.maxmind.com/app/geolitecountry
  - Download: http://geolite.maxmind.com/download/geoip/database/GeoLiteCountry/GeoIP.dat.gz

- OR -

- GeoLite City binary format database
  - Information: http://www.maxmind.com/app/geolitecity
  - Download: http://geolite.maxmind.com/download/geoip/database/GeoLiteCity.dat.gz


GeoIP API Library
--------------------
The geoip module uses MaxMind's GeoIP PHP API to interact with the databases.
You will need to download a few of these API files before enabling this module.
More information about the API can be found at http://www.maxmind.com/app/php.

To be installed in the lib directory:
- http://geolite.maxmind.com/download/geoip/api/php/geoip.inc

If using the GeoLite City database you will also need to download:
- http://geolite.maxmind.com/download/geoip/api/php/geoipcity.inc
- http://geolite.maxmind.com/download/geoip/api/php/geoipregionvars.php
